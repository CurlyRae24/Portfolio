Flyers

With these projects, I primarily used InDesign to create aestethetically pleasing formats, putting page layout skills into practice.

Examples

Scholarship Flyer

This flyer was designed to submit for the Print and Graphics Foundation Scholarship Flyer Contest. The general page layout was formatted using Adobe InDesign, with the brain image designed using image trace and brush effects in Adobe Illustrator.

GoPitt! Fest Flyer

With general layout completed using InDesign, I created this poster to promote a campus event. With these images, I used Photoshop to ensiure that they were the correct size and color mode for the printer.

Spill the Tea Flyer

This flyer was an advertisement for an event that I coordinated through my college's graphics honors society chapter. I used Illustrator to create the teacup and word bubble, then formatted the images and created my type with InDesign.

Detailed Description

These flyers required background knowledge, including the ability to adjust images to the necessary PPI and color modes for either print or digital use, along with the ability to correctly format images to bleed marks and margins in order to be viable for print.

Technologies Used:

  • Adobe InDesign
  • Adobe Illustrator
  • Adobe Photoshop
