Sunflower Riff Studios Project

2024

Project Overview

This project was created for a local music studio. I worked to combine the circular shapes of both sunflowers and vinyl records to create different versions of the logo.

Project Images

Detailed Description

To give the client a more realistic view of what their branding could look like in action, I created mockups for business cards, notepads, and letterheads. The client was happy with this branding after some font and coloring adjustments.

Technologies Used:

  • Adobe Illustrator
