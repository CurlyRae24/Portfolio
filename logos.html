Sunburst Slices

2024

Project Overview

In this project, I created a design for a hypothetical canned fruit brand entitled "Sunburst Slices.

Project Images

Detailed Description

This project was primarily made using Illustrator, specifically to create the banner effect using the warp tool. Illustrator was also used to create the logo and the nutrition label. InDesign was utiized for certain layouts. Photoshop was essential to both remove the background from the fruit images and create the mockups using the masking tool.

Technologies Used:

  • Adobe Illustrator
  • Adobe Photoshop
  • Adobe InDesign
